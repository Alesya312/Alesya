WebP

WebP, который известен своим усовершенствованным алгоритмом сжатия, позволяющим уменьшить размер изображения без ущерба для его качества.

В настоящее время формат поддерживается следующими браузерами

  • Google Chrome: WebP поддерживается браузером с версии 2.
  • Mozilla Firefox: он начал поддерживать WebP в версии 65.
  • Microsoft Edge: он начал поддерживать WebP с версии 18, но теперь Edge основан на Chromium и поддерживает WebP.
  • Safari: он начал поддерживать WebP в версии 16.5.
  • Opera: поддерживает WebP с версии 11.
  • Android Browser: поддерживает WebP с версии 4.0 (Ice Cream Sandwich).
  • Chrome для Android: работает с WebP начиная с версии 25.
  • Samsung Internet Browser: поддерживает WebP с версии 4.0.

Размер изображения 105 kB

При большем сжатии файлов становится заметна разница между .webp и .jpg. Файл в формате .webp выглядит лучше с более плавными переходами цвета. На изображении формата .jpg заметны появились заметные ореолы вокруг некоторых объектов и искажения цвета. Для данного изображения больше подходит формат .webp, но если важны четкость и мелкие детали больше подойдет формат .jpg, так как .webp заметно сглаживает картинку.
